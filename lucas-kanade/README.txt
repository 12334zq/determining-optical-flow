- To find optical flow of images use 1 of 2 commands:
    imLK(ws) - ws is window size.
    or:
    imLK - in this case, ws gets default values 15.

- To find optical flow of given video use 1 of 2 commands:
    vidLK(ws) - ws is window size.
    or:
    vidLK - in this case, ws gets default values 15.
